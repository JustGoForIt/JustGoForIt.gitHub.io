文档节点

文档节点是每个文档的根节点，文档节点只有一个子节点，即元素，成为文档元素。

  • nodeType : 9
  • nodeName : “#document”

元素节点

  • nodeType : 1
  • nodeName : DIV SPAN P
  • nodeValue : null

文本节点

  • nodeType : 3
  • nodeName : #text
  • nodeVale : 文本的值

hasChildNodes

在节点包含一个或多个节点的情况下返回true

childeNodes

  • nodeList
  • nodeList[0] <=> nodeList.item(0)

previousSibling

第一个节点的previousSibling属性值为null

nextSibling

最后一个节点的nextSibling属性为null

类数组对象

firstChild

nodeList[0]

lastChild

nodeList[nodeList.length-1]

将类数组对象转化为数组对象的方法

appendChild()

如果传入到appendChild()中的节点已经是文档的一部分了，那么该节点从原来的位置转移到新位置。

insertBefore(newNode,null/someNode)

insertBefore(newNoed,null)<=>appendChild()

replaceChild(newNode,oldNode)

返回被替换的节点，从技术上讲，被替换的节点仍然还在文档中，但它在文档中已经没有了自己的位置。

removeChild()

cloneNode()

true => 深复制
false => 浅复制
